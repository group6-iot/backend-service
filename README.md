# Installation
## 1. Ubuntu/MacOS:
```
git clone git@github.com:group6-iot/backend-service.git
cd group6-iot
bash scripts/install-linux.sh
```
## 2. Windows:
```
git clone git@github.com:group6-iot/backend-service.git
cd group6-iot
bash scripts/install-windows.sh
```

# Run Project
```
bash scripts/run.sh
```
